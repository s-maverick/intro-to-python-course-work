# Introduction to python course work

## University of Massachusetts, Amherst [DACSS 690P]

This repository serves as a comprehensive collection of foundational programming exercises completed during the Intro to Python course. It tracks the progression from basic syntax and control flow to more advanced concepts like object-oriented programming, file I/O, and data manipulation. The repository is structured to demonstrate clean coding practices, algorithmic problem-solving, and the practical application of Python in real-world scenarios.

### Tech Stack & Tools

* **Language:** Python (Core syntax, Functional & OOP paradigms).
* **Libraries:** NumPy, Pandas (Data Analysis), Matplotlib/Seaborn (Visualization).
* **Development Environment:** Jupyter Notebooks and Google Colab.
* **Testing & Version Control:** Pytest (Unit Testing), Git, GitHub.
* **Utilities:** JSON/CSV processing, Regular Expressions (RegEx).
